Skills 2026

Agentic Growth Coach — 成长教练 Skill

从 Vibe Coder 到 Agentic Engineer 的实战教练，帮助 AI 编程用户通过实战项目转型成为 Agent 系统工程师

Claude Code AI Skill Agentic Engineering 教学系统
项目概述

不是先教后做，而是边做边学

Agentic Growth Coach 是一个为 Claude Code 设计的 Skill，帮助 Vibe Coder（纯 AI 编程用户） 通过实战项目转型成为 Agentic Engineer（能够设计和构建 Agent 系统的工程师）。 它不是传统的教学工具，而是一个"在做中学"的实战教练。

双模式教学

创建模式（12 步流程）+ 分析模式（8 步流程），覆盖从 0 到 1 和从 1 到 N 的学习路径

批判式反馈

指出对错，解释为什么。二次提示机制：第一次引导思考，第二次才讲原理

知识分层

区分稳定知识（5 年+）和易变知识（需查文档），培养架构思维而非语法记忆

Agent 架构

教授 6 种基础架构模式：ReAct、Planning、Supervisor、Hierarchy、Parallel、Debate

垂直切分

Agentic 项目的稳妥实施方式，每周都有可工作的东西，避免最后才发现架构错误

质量保证

所有知识追溯权威来源（Anthropic、OpenAI、LangGraph），确保教学内容可靠

教学体系

12 步创建流程

01

需求澄清

核心问题、输入输出、成功标准

02

选择 Agent 架构模式

6 种基础架构选型

03

模块拆解

6 个标准模块：输入、LLM、规划、工具、记忆、输出

04

数据流设计

信息在系统中的流动路径

05

状态管理设计

Agent 在任何时刻"知道"的一切

06

工具选择与接口设计

搜索类、知识类、执行类、输出类

07

框架选择

LangGraph、LangChain、CrewAI、AutoGen

08

实施策略（垂直切分）

4 周计划，每周都有可工作的东西

09

AI 写代码的指挥方式

定义结构，AI 填充实现

10

Code Review 训练

检查数据流、错误处理、状态更新、模块职责

11

调试思维

状态→决策→工具调用→提示词

12

迭代规划

收集反馈、定位问题、优先级排序、小步迭代

逆向学习

8 步分析流程

通过分析现有产品学习 Agent 系统设计，从"用户视角"到"架构师视角"

A1

产品观察

入口、输出、核心能力、过程记录

A2

架构推断

根据行为推断架构模式

A3

模块边界识别

把产品拆成"黑盒模块"

A4

数据流反推

从输出倒推输入

A5

接口猜测

定义模块输入输出

A6

技术栈推测

LLM、工具、框架

A7

复刻简化版

保留核心，简化次要

A8

对比分析

通过对比发现差距

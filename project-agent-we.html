Agent_WE — 多智能体对话系统

基于 LangChain + LangGraph 的轻量化多智能体对话系统，提供情绪陪伴和工作讨论支持

LangChain LangGraph Streamlit ChromaDB DeepSeek API
项目概述

不是一个聊天机器人，而是一个智能体协作网络

Agent_WE 是一款轻量化多智能体对话系统，通过模拟不同角色的 AI 智能体进行协作对话， 为用户提供情绪价值陪伴和工作讨论支持。无需昂贵的服务器，个人电脑即可运行， 支持开源本地模型和低成本云 API，开箱即用，高度可扩展。

情绪陪伴群

夸夸师 Sunny + 捧哏大师 Buddy，提供温暖真诚的赞美和幽默风趣的对话，获得积极情绪价值

工作讨论群

产品经理 ProdMaster + 研发工程师 CodeForge，模拟真实工作场景的跨角色协作讨论

智能体管理

可视化创建和管理自定义智能体，配置专属知识库，轻松添加新角色和群聊配置

统一入口

一站式管理和访问所有应用，实时监控运行状态，一键启动，快速复制启动命令

极低成本

使用 DeepSeek API 每次对话约¥0.01，支持本地模型完全免费，本地运行零服务器成本

高度可扩展

基于 LangChain + LangGraph 构建，轻松添加新智能体角色、知识库和对话模式

对话特点

  • 智能体之间互相理解并回应彼此的话语

  • 智能识别用户参与时机，适时邀请用户加入讨论
  • 用户可随时打断并开始新话题
